## 📌 About the Project

This project is a backend application developed using FastAPI. It provides simple and efficient REST APIs that can be used for web or mobile applications.

The main goal is to understand backend development concepts like routing, request handling, and API design using Python.

FastAPI automatically generates interactive API documentation, making it easy to test and debug endpoints.

---

## ⚡ Features

- Fast and high-performance API
- Built using FastAPI framework
- RESTful architecture
- Auto-generated Swagger & ReDoc documentation
- Easy to extend and maintain
- Lightweight and beginner-friendly
